# SIG-SUAS - Sistema Integrado de Gestão do SUAS

## Sobre o Projeto

O **SIG-SUAS (Sistema Integrado de Gestão do SUAS)** é um MVP (Minimum Viable Product) concebido para a disciplina de **Gestão Ágil de Projetos e Produtos**, com o objetivo de demonstrar a aplicação das práticas de Engenharia de Requisitos, Lean Inception, Scrum e Design de Interfaces na concepção de um produto de software.

O sistema tem como propósito apoiar as atividades realizadas pelas unidades do Sitema Único de Assistência Social, sobretudo os CRAS, oferecendo uma plataforma integrada para gerenciamento dos atendimentos socioassistenciais.

O MVP contempla as seguintes funcionalidades principais:

- Cadastro e manutenção de indivíduos;
- Cadastro e manutenção de famílias;
- Registro e consulta de atendimentos;
- Visualização do prontuário do indivíduo e de sua família;
- Gestão da agenda de atendimentos;
- Emissão de documentos técnicos (relatórios, pareceres e declarações);
- Gerenciamento de usuários do sistema.

---

## Objetivo

O objetivo deste MVP é demonstrar como práticas ágeis podem ser utilizadas desde a ideação até o planejamento inicial de um produto de software, produzindo os principais artefatos necessários para o desenvolvimento de um sistema.

Não faz parte deste projeto a implementação do software. O foco desta entrega está exclusivamente na documentação e no planejamento do produto.

| Arquivo | Descrição |

| **README.md** | Apresentação do projeto e organização do repositório. |
